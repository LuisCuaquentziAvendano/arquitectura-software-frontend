# Game Hub - Frontend

## Pasos para ejecutar el proyecto
1. Instalar las dependencias
```bash
npm install
```
2. Crear el archivo .env con las variables de entorno
3. Iniciar la aplicación en desarrollo
```bash
npm run dev
```
4. Consultar la aplicación desde el navegador
```bash
http://localhost:5173/
```
NOTA: para el funcionamiento del frontend, se requiere del backend
